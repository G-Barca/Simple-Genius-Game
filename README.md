# Projeto Jogo Genius

## Jogo Básico

Projeto Simples feito com **HTML, CSS e JavaScript**. O Jogo funciona como um teste de memória, 
onde o jogador deve repetir sequências de luzes e sons emitidas pelo aparelho.
